### 一、项目介绍

基于Python的数据抓取与美食推荐系统分为前端用户端和后台管理系统。

用户端包括注册登录、查看系统推荐菜品列表、全部菜品列表、按分类筛选菜品、查看菜品统计分析、查看菜品详情、查看评论列表及发表评论、修改密码等。

- 美食列表：可以按推荐、价格两种排序方式查看美食列表，也可以按分类筛选美食列表。
- 推荐算法：根据当前登录用户浏览过的菜品信息的所属分类来推荐美食列表。
- 美食详情：查看美食详细信息，如美食价格、推荐人数等。
- 发表评论：可以在美食详情页查看评论列表及发表评论。
- 我的评论：显示我的评论历史信息。
- 编辑资料：编辑个人资料，如头像、昵称、手机号、邮箱、个人简介等。

管理系统功能包括统计分析、菜品管理、分类管理、评论管理、用户管理、数据抓取。

- 统计分析：包括统计美食总数、菜系总数、评论总数、用户总数、以柱状图的方式统计热门美食排名、以折线图的方式统计美食点击率、以饼图的方式统计热门美食分类。
- 分类管理：新增、编辑、删除及批量删除分类信息。
- 菜品管理：新增、搜索、编辑、删除及批量删除菜品信息。
- 评论管理：删除及批量删除评论信息。
- 用户管理：新增、搜索、编辑、删除及批量删除用户信息。
- 数据抓取：使用python爬虫技术，去爬取大众点评网站上的美食信息，包括美食图片、名称、价格、推荐人数、所属分类等。
